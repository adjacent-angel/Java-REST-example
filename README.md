# Java REST API Example

**Java Spring Boot REST API demonstrating backend fundamentals and clean project structure.**

This project showcases a simple RESTful API built using Java and Spring Boot, focusing on clear architecture, request handling, and CRUD operations.

## Tech Stack
- Java
- Spring Boot
- RESTful APIs
- Maven

## What I Built
- Implemented REST endpoints following standard HTTP conventions
- Structured controller and service layers for maintainability
- Demonstrated CRUD operations and request/response handling
- Organized the project to reflect real-world backend services

## Purpose
Most of my professional Java and Spring Boot work was completed in private company repositories. This project serves as a public example of my backend development style and understanding of REST API design.
